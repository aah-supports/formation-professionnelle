Module 8 — Projet de synthèse Python

Assembler toutes les briques du niveau 1 sur un cas concret.

Objectif final

  • Lire un fichier CSV en Python pur.
  • Nettoyer et valider les données.
  • Calculer des indicateurs simples.
  • Produire un résumé JSON et un graphique.

Étape 1 — Lecture CSV

import csv


def load_rows(path: str) -> list[dict]:
    with open(path, "r", encoding="utf-8") as f:
        return list(csv.DictReader(f))

Étape 2 — Nettoyage

def clean_rows(rows: list[dict]) -> list[dict]:
    cleaned = []
    for r in rows:
        if not r.get("canal") or not r.get("montant"):
            continue
        montant = float(r["montant"])
        if montant < 0:
            continue
        cleaned.append({"canal": r["canal"], "montant": montant})
    return cleaned

Étape 3 — Agrégation

def total_by_channel(rows: list[dict]) -> dict[str, float]:
    acc: dict[str, float] = {}
    for r in rows:
        canal = r["canal"]
        acc[canal] = acc.get(canal, 0.0) + r["montant"]
    return acc

Étape 4 — Export JSON

import json


def save_report(path: str, report: dict) -> None:
    with open(path, "w", encoding="utf-8") as f:
        json.dump(report, f, ensure_ascii=False, indent=2)

Étape 5 — Graphique

import matplotlib.pyplot as plt


def plot_channels(totals: dict[str, float]) -> None:
    names = list(totals.keys())
    values = list(totals.values())
    plt.bar(names, values)
    plt.title("CA par canal")
    plt.tight_layout()
    plt.show()

Orchestration

def main():
    rows = load_rows("ventes.csv")
    rows = clean_rows(rows)
    totals = total_by_channel(rows)
    save_report("rapport.json", {"totaux": totals})
    plot_channels(totals)

if __name__ == "__main__":
    main()

Contrôles minimaux

  • Fichier absent → message clair.
  • Données invalides → ligne ignorée proprement.
  • Sorties stables: même entrée, même résultat.
  • Rapport JSON lisible par un non-développeur.

Critères de réussite

  • Code exécutable de bout en bout.
  • Fonctions courtes et nommées clairement.
  • Aucune dépendance data avancée au niveau 1.
  • Synthèse compréhensible pour le métier.

Récapitulatif

  • Le niveau 1 se concentre sur Python pur et la méthode.
  • Les fondations sont prêtes pour le niveau avancé data.
  • Les bibliothèques data avancées sont traitées au niveau 3.
