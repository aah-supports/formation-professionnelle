texte = "café ☕"
raw = texte.encode("utf-8")

decoded = raw.decode("utf-8")
assert decoded == texte

Regex de base

Valider et extraire des motifs texte

import re

email = "contact@formation-pro.fr"
ok = re.match(r"^[\\w\\.-]+@[\\w\\.-]+\\.[a-z]{2,}$", email)
print(bool(ok))

texte = "Commande A-2026-015 validée"
ids = re.findall(r"[A-Z]-\\d{4}-\\d{3}", texte)
print(ids)
  • match pour validation
  • findall pour extraction
  • Commencer simple, tester souvent

Précision des flottants

Comprendre les limites de représentation

import math

print(0.1 + 0.2)                # 0.30000000000000004
print((0.1 + 0.2) == 0.3)       # False
print(math.isclose(0.1 + 0.2, 0.3))  # True
  • Comparer des floats avec tolérance
  • Éviter l'égalité stricte sur décimaux
