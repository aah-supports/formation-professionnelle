Chapitre 2 - Probabilités simples vers Bayes

Niveau Bac : raisonner avec les conditionnements.

Objectif

Savoir calculer et interpréter une probabilité conditionnelle, puis préparer l'inversion du conditionnement.

Notations de base

Rappels
  • Ω : univers des issues possibles.
  • A, B : événements.
  • Ā : complémentaire de A.
  • A ∩ B : A et B en même temps.
  • A ∪ B : A ou B.

Probabilité conditionnelle

Formule
P(A|B) = P(A ∩ B) / P(B) avec P(B) ≠ 0
  • "Sachant B", on restreint l'univers à B.
  • Attention : P(A|B) n'est pas P(B|A).

Arbre de probabilités

Méthode
  • Écrire les branches à chaque étape.
  • Multiplier sur une branche complète.
  • Additionner les branches compatibles.
Exemple

Test médical : Test+ / Test- selon Malade / Non malade.

Probabilités totales

Idée

Si B et B̄ forment une partition :

P(A) = P(A|B)P(B) + P(A|B̄)P(B̄)

Très utile avant d'inverser un conditionnement.

Préparer Bayes sans surcharge

Question type

"Le test est positif : quelle est la probabilité d'être malade ?"

  • On cherche un conditionnement inversé.
  • On ne doit pas oublier la fréquence de base de la maladie.
  • Le tableau ou l'arbre évite les erreurs.

Exemple chiffré simple

Données
MaladeNon malade
Population1%99%
Test positif95%5%

Lecture guidée : un test positif ne signifie pas automatiquement "forte probabilité d'être malade".

Exercice guidé

À faire
  • Construire l'arbre correspondant à l'exemple.
  • Calculer la probabilité d'avoir un test positif.
  • Interpréter qualitativement la probabilité recherchée après test positif.

Projet final - 1h

Application complète
  • Analyser une série de notes avec les indicateurs de dispersion.
  • Construire un arbre de probabilités sur un cas de test de dépistage.
  • Rédiger une conclusion argumentée et lisible.

Résumé du chapitre

À retenir
  • Conditionner, c'est restreindre l'univers.
  • Arbres et tableaux limitent les confusions.
  • On est prêt à aborder le théorème de Bayes avec sérénité.
