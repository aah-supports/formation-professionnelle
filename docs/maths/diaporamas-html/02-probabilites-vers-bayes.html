Chapitre 2 - Probabilités simples vers Bayes

Raisonner avec les conditionnements.

Objectif

Passer de la probabilité conditionnelle à une application complète de la probabilité totale puis de Bayes.

Notations de base

Rappels
  • Ω : univers des issues possibles.
  • M : événement "malade" ; non M : complémentaire.
  • + : événement "test positif".
  • M ∩ + : malade et test positif.

Méthode en 5 étapes

Procédure
  1. Poser les données de fréquence de base et de test.
  2. Construire l'arbre (M / non M puis + / -).
  3. Calculer les probabilités de branches complètes.
  4. Appliquer la probabilité totale pour obtenir P(+).
  5. Appliquer Bayes pour obtenir P(M|+).

Exemple - Données

Hypothèses
Malade (M)Non malade (non M)
Population1%99%
Test positif95%5%

Objectif : calculer P(+) puis P(M|+).

Arbre de probabilités (joli format)

Visualisation
Population +-- M (1%) | +-- + (95%) => P(M ∩ +) = 0,01 x 0,95 = 0,0095 | \-- - (5%) => P(M ∩ -) = 0,01 x 0,05 = 0,0005 \-- non M (99%) +-- + (5%) => P(non M ∩ +) = 0,99 x 0,05 = 0,0495 \-- - (95%) => P(non M ∩ -) = 0,99 x 0,95 = 0,9405

Correction de l'exemple - Probabilité totale

Étape 1
P(+) = P(+|M)P(M) + P(+|non M)P(non M)
P(+) = 0,95×0,01 + 0,05×0,99 = 0,059
Résultat : P(+)=5,9%

Correction de l'exemple - Bayes

Étape 2
P(M|+) = [P(+|M)P(M)] / P(+)
P(M|+) = (0,95×0,01) / 0,059 = 0,161
Résultat : P(M|+) ≈ 16,1%

Conclusion : un test positif n'implique pas une probabilité majoritaire d'être malade.

Vérification sur 10 000 personnes

Contrôle intuitif
  • Malades : 100 -> positifs vrais : 95
  • Non malades : 9 900 -> faux positifs : 495
  • Total positifs : 590
  • P(M|+) = 95/590 ≈ 16,1%

Formules à retenir

Probabilité totale
P(A)=P(A|B)P(B)+P(A|B barre)P(B barre)
Bayes
P(B|A)=[P(A|B)P(B)]/P(A)

Exercice guidé

Application
  • Refaire le même exemple avec un faux positif de 2%.
  • Comparer la nouvelle valeur de P(M|+).
  • Conclure sur l'impact des faux positifs.

Projet final - 1h

Application complète
  • Analyser une série de notes avec les indicateurs de dispersion.
  • Construire un arbre de probabilités sur un cas de test de dépistage.
  • Rédiger une conclusion argumentée et lisible.

Résumé du chapitre

À retenir
  • Arbre -> branches -> probabilité totale -> Bayes.
  • Ne pas confondre P(M|+) et P(+|M).
  • La fréquence de base influence fortement l'interprétation.
