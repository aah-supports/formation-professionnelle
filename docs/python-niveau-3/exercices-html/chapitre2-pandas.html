Chapitre 2 — Nettoyage et transformation avec pandas

Objectif: transformer un export brut en table analytique fiable.

Exercice 2.1 — Nettoyage typage et valeurs manquantes Intermédiaire

  1. Charger un CSV contenant date_commande, montant, region.
  2. Parser les dates, convertir montant en float.
  3. Supprimer les lignes sans région et imputer les montants manquants par médiane régionale.
Corrigé 
import pandas as pd


df = pd.read_csv("ventes_brut.csv")

df["date_commande"] = pd.to_datetime(df["date_commande"], errors="coerce")
df["montant"] = (
    df["montant"].astype(str)
    .str.replace(" ", "", regex=False)
    .str.replace(",", ".", regex=False)
)
df["montant"] = pd.to_numeric(df["montant"], errors="coerce")

df = df.dropna(subset=["region"])
df["montant"] = df.groupby("region")["montant"].transform(
    lambda s: s.fillna(s.median())
)

Exercice 2.2 — Pipeline reproductible Intermédiaire

  1. Écrire une fonction prepare_ventes(df).
  2. Chaîner les étapes avec .pipe().
  3. Retourner un DataFrame trié par date et id commande.
Corrigé 
def standardize_columns(df):
    df = df.copy()
    df.columns = [c.strip().lower() for c in df.columns]
    return df


def prepare_dates(df):
    df = df.copy()
    df["date_commande"] = pd.to_datetime(df["date_commande"], errors="coerce")
    return df


def prepare_ventes(df):
    return (
        df.pipe(standardize_columns)
          .pipe(prepare_dates)
          .dropna(subset=["id_commande", "date_commande"])
          .sort_values(["date_commande", "id_commande"])
          .reset_index(drop=True)
    )

Exercice 2.3 — Contrôle qualité tabulaire Avancé

Produire un tableau QA avec, par colonne: type, nombre de NA, taux NA, nombre de valeurs distinctes.

Corrigé 
def quality_report(df: pd.DataFrame) -> pd.DataFrame:
    n = len(df)
    report = pd.DataFrame({
        "dtype": df.dtypes.astype(str),
        "na_count": df.isna().sum(),
        "na_rate": (df.isna().sum() / n).round(4),
        "n_unique": df.nunique(dropna=True),
    })
    return report.sort_values("na_rate", ascending=False)

qa = quality_report(df)
print(qa)
