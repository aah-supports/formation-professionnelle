Exercice 6.2 — Benchmark avant/après Avancé

  1. Comparer version baseline vs version optimisée.
  2. Mesurer temps moyen sur 10 runs.
  3. Documenter le gain et les compromis.
Corrigé 
from time import perf_counter


def bench(fn, *args, n=10):
    times = []
    for _ in range(n):
        t0 = perf_counter()
        fn(*args)
        times.append(perf_counter() - t0)
    return sum(times) / len(times)

base_t = bench(run_pipeline_base, df)
opt_t = bench(run_pipeline_opt, df)
print(f"Baseline: {base_t:.3f}s | Opt: {opt_t:.3f}s | Gain: {(1 - opt_t/base_t):.1%}")

Exercice 6.3 — Pipeline CI/CD minimal Avancé

Construire un workflow GitHub Actions avec étapes:

  • lint + typing
  • tests
  • build package
  • publication sur tag
Corrigé 
name: ci
on: [push, pull_request]

jobs:
  quality:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: actions/setup-python@v5
        with: { python-version: '3.12' }
      - run: pip install -r requirements.txt
      - run: ruff check src tests
      - run: mypy src
      - run: pytest -q --maxfail=1
