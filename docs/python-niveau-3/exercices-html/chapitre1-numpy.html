Exercice 1.1 — Vectorisation d'un calcul de marge Intermédiaire

Énoncé: on vous donne trois tableaux NumPy de même taille: prix_vente, cout_unitaire, quantite.

  1. Calculer la marge ligne par ligne sans boucle explicite.
  2. Calculer la marge totale.
  3. Retourner aussi le taux de marge global.

Critère de réussite: aucun for, uniquement opérations vectorisées.

Corrigé 
import numpy as np

prix_vente = np.array([120, 80, 150, 60], dtype=float)
cout_unitaire = np.array([70, 55, 90, 40], dtype=float)
quantite = np.array([10, 8, 6, 12], dtype=float)

marge_ligne = (prix_vente - cout_unitaire) * quantite
marge_totale = marge_ligne.sum()
ca_total = (prix_vente * quantite).sum()
taux_marge = marge_totale / ca_total

print(marge_ligne, marge_totale, taux_marge)

Exercice 1.2 — Normalisation par colonne Intermédiaire

Énoncé: à partir d'une matrice X de forme (n_lignes, n_features), standardiser chaque colonne avec un z-score.

  1. Calculer moyenne et écart-type sur le bon axe.
  2. Protéger le calcul contre les divisions par zéro.
  3. Vérifier que chaque colonne normalisée a une moyenne proche de 0.
Corrigé 
import numpy as np

X = np.array([
    [10, 200, 5],
    [12, 220, 5],
    [14, 210, 5],
    [16, 230, 5],
], dtype=float)

mu = X.mean(axis=0)
sigma = X.std(axis=0)
sigma_safe = np.where(sigma == 0, 1.0, sigma)

Xz = (X - mu) / sigma_safe

print("moyennes:", Xz.mean(axis=0))
print("std:", Xz.std(axis=0))

Exercice 1.3 — Segmentation par masque booléen Avancé

Énoncé: on veut isoler les commandes « à risque »: montant > 1000 et délai > 5 jours.

  1. Construire un masque booléen combiné.
  2. Extraire les identifiants concernés.
  3. Calculer la part des commandes à risque.
Corrigé 
import numpy as np

order_id = np.array([101, 102, 103, 104, 105, 106])
montant = np.array([450, 1200, 990, 1600, 800, 1800], dtype=float)
delai = np.array([2, 7, 4, 8, 3, 6], dtype=int)

mask_risque = (montant > 1000) & (delai > 5)
orders_risque = order_id[mask_risque]
part_risque = mask_risque.mean()

print(orders_risque)
print(f"Part risque: {part_risque:.2%}")
