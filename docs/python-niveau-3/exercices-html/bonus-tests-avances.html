Exercice B2 — Mock de dépendance externe Bonus

  1. Mocker un client HTTP dans un service de synchronisation.
  2. Vérifier que l'appel est fait avec le bon payload.
  3. Tester le cas d'erreur réseau et le retry.
Corrigé 
from unittest.mock import Mock
import pytest

def test_push_payload_once():
    client = Mock()
    svc = SyncService(client)

    svc.push({"order_id": 1})
    client.post.assert_called_once_with("/orders", json={"order_id": 1})


def test_push_retry_on_error():
    client = Mock()
    client.post.side_effect = [TimeoutError(), None]
    svc = SyncService(client, retries=1)

    svc.push({"order_id": 1})
    assert client.post.call_count == 2

Exercice B3 — Tests async avec timeout Bonus

  1. Écrire un test async qui impose un timeout strict.
  2. Tester la branche timeout et la branche succès.
  3. Tracer la durée d'exécution pour éviter les tests lents.
Corrigé 
import pytest

@pytest.mark.asyncio
async def test_fetch_success(async_client):
    result = await async_client.fetch(timeout=0.2)
    assert result["status"] == "ok"


@pytest.mark.asyncio
async def test_fetch_timeout(async_client):
    with pytest.raises(TimeoutError):
        await async_client.fetch(timeout=0.001)
