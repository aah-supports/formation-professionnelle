Exercice 3.2 — KPI mensuels et variation MoM Avancé

  1. Agréger CA, nb commandes, panier moyen par region et mois.
  2. Calculer variation mensuelle du CA.
  3. Identifier les régions en baisse 2 mois consécutifs.
Corrigé 
kpi = (
    merged.groupby(["region", "mois"], as_index=False)
          .agg(
              ca_total=("montant", "sum"),
              nb_commandes=("id_commande", "nunique"),
          )
)
kpi["panier_moyen"] = kpi["ca_total"] / kpi["nb_commandes"]

kpi = kpi.sort_values(["region", "mois"])
kpi["ca_prev"] = kpi.groupby("region")["ca_total"].shift(1)
kpi["mom_pct"] = (kpi["ca_total"] - kpi["ca_prev"]) / kpi["ca_prev"]

kpi["baisse"] = kpi["mom_pct"] < 0
kpi["baisse_2x"] = kpi.groupby("region")["baisse"].transform(lambda s: s & s.shift(1).fillna(False))

Exercice 3.3 — Contrôle de cohérence KPI Avancé

Produire une table de validation avec:

  • KPI manquants, valeurs négatives, divisions invalides.
  • Écart vs objectif > 20%.
  • Statut final: OK, ALERTE, ERREUR.
Corrigé 
qc = kpi.copy()
qc["target_gap_pct"] = (qc["ca_total"] - qc["objectif_ca"]) / qc["objectif_ca"]

conditions = [
    qc[["ca_total", "nb_commandes", "panier_moyen"]].isna().any(axis=1),
    (qc["ca_total"] < 0) | (qc["nb_commandes"] <= 0),
    qc["target_gap_pct"].abs() > 0.20,
]
choices = ["ERREUR", "ERREUR", "ALERTE"]

import numpy as np
qc["statut"] = np.select(conditions, choices, default="OK")
print(qc[["region", "mois", "statut", "target_gap_pct"]])
