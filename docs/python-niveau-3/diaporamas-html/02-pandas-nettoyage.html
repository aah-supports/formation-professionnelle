Chapitre 2 — Nettoyage et transformation avec pandas

Passer de données brutes à une table analytique fiable.

Objectifs du chapitre

  • Définir un contrat de schéma explicite.
  • Nettoyer valeurs manquantes, doublons et incohérences.
  • Sécuriser jointures et agrégations.
  • Produire un pipeline lisible et relançable.

Import robuste

import pandas as pd

df = pd.read_csv(
    "ventes.csv",
    sep=",",
    encoding="utf-8",
    parse_dates=["date_commande"],
    dtype={"id_commande": "string", "canal": "string"}
)

Contrat de schéma minimal

required = {
    "id_commande", "date_commande", "prix_net", "canal"
}
missing = required - set(df.columns)
if missing:
    raise ValueError(f"colonnes manquantes: {sorted(missing)}")

Gestion des valeurs manquantes

na_ratio = (df.isna().sum() / len(df)).sort_values(ascending=False)

# stratégie explicite
df["canal"] = df["canal"].fillna("inconnu")
df = df.dropna(subset=["id_commande", "date_commande"])

Doublons et unicité

before = len(df)
df = df.drop_duplicates(subset=["id_commande"], keep="last")
after = len(df)
print(f"doublons supprimés: {before - after}")

Typage explicite

df = df.convert_dtypes()

df["prix_net"] = pd.to_numeric(df["prix_net"], errors="coerce")
df["qte"] = pd.to_numeric(df["qte"], errors="coerce").astype("Int64")

df = df.dropna(subset=["prix_net", "qte"])

Nettoyage texte standard

df["ville"] = (
    df["ville"]
      .str.normalize("NFKC")
      .str.strip()
      .str.lower()
      .str.replace(r"\s+", " ", regex=True)
)

Dates et fuseaux

df["date_commande"] = pd.to_datetime(df["date_commande"], utc=True)
df["date_paris"] = df["date_commande"].dt.tz_convert("Europe/Paris")

df["mois"] = df["date_paris"].dt.to_period("M")

Jointures sécurisées

orders = df
clients = pd.read_csv("clients.csv", dtype={"client_id": "string"})

merged = orders.merge(
    clients,
    on="client_id",
    how="left",
    validate="many_to_one"
)

groupby : agg vs transform

kpi = df.groupby("canal", as_index=False).agg(
    ca=("prix_net", "sum"),
    commandes=("id_commande", "nunique")
)

df["zscore_prix"] = (
    df["prix_net"] - df["prix_net"].mean()
) / df["prix_net"].std()

Reshape : pivot et melt

wide = df.pivot_table(
    index="mois", columns="canal", values="prix_net", aggfunc="sum", fill_value=0
)
long = wide.reset_index().melt(id_vars="mois", var_name="canal", value_name="ca")

Pipeline lisible avec pipe

def drop_negative_prices(frame: pd.DataFrame) -> pd.DataFrame:
    return frame[frame["prix_net"] >= 0]

clean = (
    df
    .pipe(drop_negative_prices)
    .assign(ca=lambda d: d["prix_net"] * d["qte"])
)

Contrôles qualité avant publication

  • Volume de lignes attendu (tolérance documentée).
  • Taux de NA par colonne critique.
  • Unicité des clés fonctionnelles.
  • Cohérence des agrégats vs source transactionnelle.

Récapitulatif

  • pandas sert d'abord à garantir la qualité de la donnée.
  • Un pipeline explicite vaut mieux qu'un notebook opaque.
  • Jointures validées et schéma contractuel évitent les erreurs métiers.
