Chapitre 3 — Async/await

Objectif: paralléliser correctement les tâches I/O, gérer le timeout et contrôler la concurrence.

Exercice 3.1 — Parallélisation I/O Intermédiaire

  1. Simuler 30 appels réseau avec asyncio.sleep.
  2. Exécuter en parallèle via asyncio.gather.
  3. Comparer la durée avec une version séquentielle.
Corrigé 
import asyncio
import time

async def fake_io(i: int) -> int:
    await asyncio.sleep(0.05)
    return i * 2

async def parallel_run():
    tasks = [fake_io(i) for i in range(30)]
    return await asyncio.gather(*tasks)

async def sequential_run():
    out = []
    for i in range(30):
        out.append(await fake_io(i))
    return out

async def main():
    t0 = time.perf_counter()
    await sequential_run()
    t1 = time.perf_counter()
    await parallel_run()
    t2 = time.perf_counter()
    print(f"seq={t1-t0:.3f}s | par={t2-t1:.3f}s")

asyncio.run(main())

Exercice 3.2 — Timeout et gestion d'erreurs Avancé

  1. Encadrer un appel async avec asyncio.wait_for.
  2. Retourner un statut standardisé (ok/timeout/error).
  3. Journaliser le motif d'échec.
Corrigé 
import asyncio

async def unstable_call(delay: float):
    await asyncio.sleep(delay)
    return {"status": "ok"}

async def safe_call(delay: float, timeout: float = 0.1):
    try:
        result = await asyncio.wait_for(unstable_call(delay), timeout=timeout)
        return result
    except asyncio.TimeoutError:
        return {"status": "timeout"}
    except Exception as exc:
        return {"status": "error", "detail": str(exc)}

Exercice 3.3 — Limiter la concurrence Avancé

  1. Utiliser un Semaphore de taille 5.
  2. Lancer 50 tâches async.
  3. Vérifier que le traitement reste stable (pas de saturation).
Corrigé 
import asyncio

sem = asyncio.Semaphore(5)

async def bounded_call(i: int) -> int:
    async with sem:
        await asyncio.sleep(0.03)
        return i

async def run_batch():
    tasks = [asyncio.create_task(bounded_call(i)) for i in range(50)]
    return await asyncio.gather(*tasks)

if __name__ == "__main__":
    out = asyncio.run(run_batch())
    print(len(out), out[:5])
