Chapitre 2 — Décorateurs

Objectif: construire des décorateurs fiables, composables et lisibles.

Exercice 2.1 — Décorateur de mesure de durée Intermédiaire

  1. Créer @log_duration avec functools.wraps.
  2. Afficher le nom de la fonction et sa durée.
  3. Vérifier que la docstring reste intacte.
Corrigé 
import time
from functools import wraps


def log_duration(fn):
    @wraps(fn)
    def wrapper(*args, **kwargs):
        t0 = time.perf_counter()
        out = fn(*args, **kwargs)
        dt = time.perf_counter() - t0
        print(f"{fn.__name__} took {dt:.4f}s")
        return out
    return wrapper

@log_duration
def compute(n: int) -> int:
    """Calcule une somme simple."""
    return sum(range(n))

Exercice 2.2 — Validation métier par décorateur Avancé

  1. Créer @validate_amount pour bloquer les montants négatifs.
  2. Renvoyer une erreur explicite.
  3. Ajouter un test qui valide le comportement.
Corrigé 
from functools import wraps


def validate_amount(fn):
    @wraps(fn)
    def wrapper(amount: float, *args, **kwargs):
        if amount < 0:
            raise ValueError("Montant négatif interdit")
        return fn(amount, *args, **kwargs)
    return wrapper

@validate_amount
def apply_payment(amount: float, fee: float = 0.0) -> float:
    return amount - fee

Exercice 2.3 — Composition de décorateurs Avancé

  1. Composer @log_duration et @retry.
  2. Vérifier l'ordre d'exécution en cas d'échec intermittent.
  3. Limiter le nombre de tentatives.
Corrigé 
from functools import wraps


def retry(max_attempts: int = 3):
    def deco(fn):
        @wraps(fn)
        def wrapper(*args, **kwargs):
            last_exc = None
            for _ in range(max_attempts):
                try:
                    return fn(*args, **kwargs)
                except Exception as exc:
                    last_exc = exc
            raise last_exc
        return wrapper
    return deco

@log_duration
@retry(max_attempts=2)
def flaky_call(state: dict):
    state["count"] += 1
    if state["count"] == 1:
        raise RuntimeError("temporary error")
    return "ok"
