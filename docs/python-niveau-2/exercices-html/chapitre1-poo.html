Exercice 1.1 — Modèle Commande robuste Intermédiaire

  1. Créer Client, LigneCommande, Commande avec dataclass.
  2. Empêcher les quantités nulles ou négatives.
  3. Exposer une propriété total_ttc calculée.
Corrigé 
from dataclasses import dataclass, field

@dataclass(frozen=True)
class Client:
    id: str
    nom: str

@dataclass(frozen=True)
class LigneCommande:
    sku: str
    quantite: int
    prix_unitaire: float

    def __post_init__(self):
        if self.quantite <= 0:
            raise ValueError("quantité invalide")
        if self.prix_unitaire < 0:
            raise ValueError("prix invalide")

    @property
    def total(self) -> float:
        return self.quantite * self.prix_unitaire

@dataclass
class Commande:
    id: str
    client: Client
    lignes: list[LigneCommande] = field(default_factory=list)
    taux_tva: float = 0.2

    @property
    def total_ht(self) -> float:
        return sum(l.total for l in self.lignes)

    @property
    def total_ttc(self) -> float:
        return self.total_ht * (1 + self.taux_tva)

Exercice 1.2 — Composition vs héritage Avancé

Refactorer une hiérarchie rigide User -> PremiumUser -> EnterpriseUser en composition de rôles.

  1. Créer une abstraction de politique de remise.
  2. Injecter la politique dans la classe Panier.
  3. Supprimer la logique conditionnelle liée au type utilisateur.
Corrigé 
from typing import Protocol

class DiscountPolicy(Protocol):
    def apply(self, amount: float) -> float: ...

class NoDiscount:
    def apply(self, amount: float) -> float:
        return amount

class PremiumDiscount:
    def apply(self, amount: float) -> float:
        return amount * 0.9

class Panier:
    def __init__(self, policy: DiscountPolicy) -> None:
        self.policy = policy

    def total(self, amount: float) -> float:
        return self.policy.apply(amount)

Exercice 1.3 — Tests unitaires sur invariants Avancé

  1. Tester la création de ligne valide.
  2. Tester les exceptions sur quantité/prix invalides.
  3. Tester le total TTC sur une commande multi-lignes.
Corrigé 
import pytest


def test_ligne_valide():
    l = LigneCommande("SKU-1", 2, 10.0)
    assert l.total == 20.0


def test_ligne_quantite_invalide():
    with pytest.raises(ValueError):
        LigneCommande("SKU-1", 0, 10.0)


def test_commande_total_ttc():
    c = Commande(
        id="CMD-1",
        client=Client("C1", "Alice"),
        lignes=[LigneCommande("A", 2, 10.0), LigneCommande("B", 1, 5.0)],
        taux_tva=0.2,
    )
    assert c.total_ht == 25.0
    assert c.total_ttc == 30.0
