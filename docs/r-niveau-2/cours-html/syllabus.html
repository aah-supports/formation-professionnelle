R Niveau II

R Niveau II — Syllabus

Exploitation de données et restitution avec R

Durée : 2 jours (14 heures) Format : Présentiel ou distanciel synchrone

Description

Cette formation vise la production d'analyses opérationnelles fiables avec R : extraction de données, normalisation de fichiers tableur, requêtes avancées, contrôles qualité, statistiques descriptives et restitution métier. Le parcours suit un cas fil rouge sur deux jours.

Public cible

  • Analystes ayant déjà les bases de R
  • Profils data devant exploiter des sources SQL et tableur
  • Chargés d'études qui construisent des KPI pour la direction
  • Professionnels en charge de fiabiliser la chaîne de traitement des données

Prérequis

  • Compétences équivalentes à R Niveau I
  • Aisance avec data.frame, filtres et agrégations simples
  • Connaissances de base en lecture de tableaux de données
  • Notions SQL utiles mais non obligatoires

Objectifs pédagogiques

  1. Extraire des données depuis une base avec R et dbplyr
  2. Importer et normaliser des fichiers Excel/tableur
  3. Construire des requêtes avancées avec jointures et segmentation
  4. Automatiser des extractions par paramétrage de scripts
  5. Contrôler la qualité des données avant publication
  6. Réaliser des statistiques descriptives par segments métier
  7. Produire une restitution opérationnelle orientée décision
  8. Livrer un projet fil rouge complet de l'extraction à la synthèse

Programme

Jour 1 — Connexion, extraction et requêtes

Module Contenu
1. Bases de données avec R Connexion DBI, extraction avec dplyr/dbplyr, requêtes paramétrées
2. Extraction Excel / tableur Import multi-feuilles, normalisation tidyr, contrôles de structure
3. Requêtes avancées I Jointures, cardinalité, segmentation métier, tables analytiques
4. Requêtes avancées II Automatisation de scénarios, gestion d'erreurs, consolidation des sorties

Jour 2 — Qualité, statistiques et restitution

Module Contenu
5. Qualité des données Contrôles NA/doublons/types, consolidation multi-sources, rapport qualité
6. Statistiques descriptives pratiques Mesures de tendance/dispersions, synthèses par groupes, interprétation
7. Restitution métier KPI, visualisation de synthèse, recommandations actionnables
8. Projet fil rouge Chaîne complète: extraction, requêtes, qualité, stats, restitution

Module bonus (hors timing) : Visualisation avancée.
