Module 5 : Contrôles qualité et consolidation

Exercice 5.1 : Profil qualité rapide Difficulté: Facile

Énoncé : Sur base_analyse, calculez pour chaque colonne :

  1. Le nombre de valeurs manquantes
  2. Le pourcentage de valeurs manquantes
  3. Le nombre de valeurs distinctes
Solution 
library(dplyr)
library(tidyr)

profil_qc <- base_analyse |>
  summarise(across(
    everything(),
    list(
      na_n = ~ sum(is.na(.x)),
      na_pct = ~ round(mean(is.na(.x)) * 100, 2),
      n_distinct = ~ n_distinct(.x)
    )
  )) |>
  pivot_longer(everything(), names_to = "metrique", values_to = "valeur")

profil_qc

Exercice 5.2 : Détection de doublons Difficulté: Intermédiaire

Énoncé : Identifiez les commandes dupliquées sur la clé (id_commande, date).

  1. Compter les occurrences par clé
  2. Ne garder que celles avec plus d'une occurrence
  3. Afficher les lignes concernées
Solution 
dups_keys <- base_analyse |>
  count(id_commande, date, name = "n") |>
  filter(n > 1)

dups_lignes <- base_analyse |>
  inner_join(dups_keys |> select(id_commande, date), by = c("id_commande", "date")) |>
  arrange(id_commande, date)

dups_lignes

Exercice 5.3 : Réconciliation base vs Excel Difficulté: Avancé

Énoncé : Comparez les montants mensuels entre :

  • ca_base issu des transactions SQL
  • ca_excel issu d'un fichier de reporting

Objectif : produire un tableau de différentiel (ecart, ecart_pct).

Solution 
reco <- ca_base |>
  full_join(ca_excel, by = c("mois", "region")) |>
  mutate(
    ca_base = coalesce(ca_base, 0),
    ca_excel = coalesce(ca_excel, 0),
    ecart = ca_base - ca_excel,
    ecart_pct = if_else(ca_excel == 0, NA_real_, ecart / ca_excel)
  ) |>
  arrange(desc(abs(ecart)))

reco

Module 6 : Statistiques descriptives pratiques

Exercice 6.1 : KPI de tendance centrale et dispersion Difficulté: Intermédiaire

Énoncé : Sur la variable montant_commande, calculez :

  • moyenne, médiane, p25, p75
  • IQR, écart-type, coefficient de variation
Solution 
kpi_dispersion <- base_analyse |>
  summarise(
    moyenne = mean(montant_commande, na.rm = TRUE),
    mediane = median(montant_commande, na.rm = TRUE),
    p25 = quantile(montant_commande, 0.25, na.rm = TRUE),
    p75 = quantile(montant_commande, 0.75, na.rm = TRUE),
    iqr = IQR(montant_commande, na.rm = TRUE),
    ecart_type = sd(montant_commande, na.rm = TRUE),
    coef_variation = ecart_type / moyenne
  )

kpi_dispersion

Exercice 6.2 : Analyse par segment client Difficulté: Intermédiaire

Énoncé : Produisez un tableau descriptif par segment avec :

  1. Nombre de clients
  2. CA total
  3. Panier moyen
  4. Médiane panier
  5. Part de CA (%)
Solution 
desc_segment <- base_analyse |>
  group_by(segment) |>
  summarise(
    nb_clients = n_distinct(client_id),
    ca_total = sum(montant_commande, na.rm = TRUE),
    panier_moyen = mean(montant_commande, na.rm = TRUE),
    panier_mediane = median(montant_commande, na.rm = TRUE),
    .groups = "drop"
  ) |>
  mutate(part_ca = ca_total / sum(ca_total)) |>
  arrange(desc(ca_total))

desc_segment

Exercice 6.3 : Indicateurs temporels pratiques Difficulté: Avancé

Énoncé : Construisez un tableau mensuel avec :

  • CA du mois
  • variation MoM (%)
  • cumul annuel
  • moyenne mobile 3 mois
Solution 
serie_mensuelle <- base_analyse |>
  mutate(mois = as.Date(format(date, "%Y-%m-01"))) |>
  group_by(mois) |>
  summarise(ca = sum(montant_commande, na.rm = TRUE), .groups = "drop") |>
  arrange(mois) |>
  mutate(
    ca_mois_prec = lag(ca),
    var_mom = (ca - ca_mois_prec) / ca_mois_prec,
    cumul_annuel = cumsum(ca),
    mm3 = (lag(ca, 2) + lag(ca, 1) + ca) / 3
  )

serie_mensuelle

Module 7 : Restitution opérationnelle

Exercice 7.1 : Tableau de pilotage avec gt Difficulté: Intermédiaire

Énoncé : À partir de desc_segment, produire un tableau lisible :

  1. Format monétaire pour CA et panier
  2. Format pourcentage pour part de CA
  3. Mise en évidence du segment avec le CA max
Solution 
library(gt)

tbl_pilotage <- desc_segment |>
  gt() |>
  fmt_currency(columns = c(ca_total, panier_moyen, panier_mediane), currency = "EUR") |>
  fmt_percent(columns = part_ca, decimals = 1) |>
  tab_style(
    style = list(cell_fill(color = "#d4edda"), cell_text(weight = "bold")),
    locations = cells_body(rows = ca_total == max(ca_total))
  )

tbl_pilotage

Exercice 7.2 : Graphiques descriptifs utiles Difficulté: Intermédiaire

Énoncé : Créer deux graphiques :

  • CA mensuel (ligne)
  • Répartition du CA par segment (barres)

Avec un titre métier explicite et theme_minimal().

Solution 
library(ggplot2)

p1 <- serie_mensuelle |>
  ggplot(aes(mois, ca)) +
  geom_line(linewidth = 1.1, color = "#0984e3") +
  geom_point(size = 2, color = "#0984e3") +
  labs(
    title = "Évolution mensuelle du chiffre d'affaires",
    x = "Mois", y = "CA"
  ) +
  theme_minimal()

p2 <- desc_segment |>
  ggplot(aes(reorder(segment, ca_total), ca_total, fill = segment)) +
  geom_col(show.legend = FALSE) +
  coord_flip() +
  labs(
    title = "Contribution au CA par segment client",
    x = "Segment", y = "CA"
  ) +
  theme_minimal()

p1
p2
