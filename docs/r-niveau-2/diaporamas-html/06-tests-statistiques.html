Tendance centrale

kpi_centre <- df |>
  summarise(
    moyenne = mean(montant, na.rm = TRUE),
    mediane = median(montant, na.rm = TRUE),
    moyenne_tronquee = mean(montant, trim = 0.1, na.rm = TRUE)
  )
3 / 10

Dispersion et stabilité

kpi_dispersion <- df |>
  summarise(
    sd = sd(montant, na.rm = TRUE),
    iqr = IQR(montant, na.rm = TRUE),
    coef_variation = sd / mean(montant, na.rm = TRUE)
  )
4 / 10

Descriptif catégoriel

desc_segment <- df |>
  group_by(segment) |>
  summarise(
    nb_clients = n_distinct(client_id),
    ca_total = sum(montant, na.rm = TRUE),
    panier_mediane = median(montant, na.rm = TRUE),
    .groups = "drop"
  ) |>
  mutate(part_ca = ca_total / sum(ca_total))
5 / 10

Séries temporelles descriptives

serie_mensuelle <- df |>
  group_by(mois) |>
  summarise(ca = sum(montant, na.rm = TRUE), .groups = "drop") |>
  arrange(mois) |>
  mutate(
    ca_mois_prec = lag(ca),
    var_mom = (ca - ca_mois_prec) / ca_mois_prec,
    cumul_annuel = cumsum(ca)
  )

Quels indicateurs retenir ?

  • Global : CA, volume, panier moyen
  • Segment : contribution et dispersion
  • Temps : tendance et ruptures
  • Qualité : part de données manquantes

Interprétation métier
