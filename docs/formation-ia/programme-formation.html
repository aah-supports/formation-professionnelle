Formation IA — Programme de formation (format prospect)

Parcours : GenAI, LLM, RAG et IA agentique Durée : 10 jours (57h15) Format : Présentiel, distanciel synchrone ou hybride

Description générale

Programme professionnel centré sur les usages concrets de l'IA générative : LLM, prompting, intégration API, RAG et IA agentique. La formation alterne apports méthodologiques, démonstrations guidées et ateliers applicatifs pour produire des résultats directement exploitables en entreprise.

Durée

10 jours (57h15), organisables en format intensif ou fractionné selon le contexte client.

Horaires

9h00-12h30 et 13h30-17h00 (adaptables en intra selon les contraintes opérationnelles).

Rythme

  • Intensif : 2 semaines consécutives.
  • Progressif : 1 à 2 jours par semaine sur 5 à 10 semaines.

Prérequis

  • Bases Python et API REST.
  • Manipulation de données textuelles/JSON.
  • Aisance terminal, Git et logique de débogage.

Certification ou examen final

Évaluation finale sur cas fil rouge : conception, implémentation et soutenance d'un prototype IA. Délivrance d'une attestation de compétences avec critères d'évaluation explicites.

Matériel nécessaire

  • Ordinateur portable (16 Go RAM recommandés).
  • Connexion Internet stable.
  • Compte GitHub et accès environnement de développement.
  • Accès API LLM (clé fournie par l'organisation ou le participant).

Modalité

Présentiel, distanciel synchrone ou format hybride. Sessions adaptées aux équipes techniques, produit et innovation.
