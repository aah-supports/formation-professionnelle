R Niveau I

R Niveau I — Syllabus

Initiation au langage R

Durée : 2 jours (14 heures) Format : Présentiel ou distanciel synchrone

Description

Formation de deux jours pour découvrir R et RStudio. Les stagiaires acquièrent les bases pour manipuler une session R, distinguer les objets du langage et mettre en œuvre les fonctions essentielles pour explorer et transformer un jeu de données.

Public cible

  • Analystes de données débutants en R
  • Statisticiens migrant depuis Excel, SPSS ou SAS
  • Étudiants en data science, biologie, économie, sciences sociales
  • Professionnels souhaitant automatiser des traitements de données

Prérequis

  • Notions de statistique descriptive (moyenne, médiane, écart-type)
  • Aisance informatique de base
  • Aucune expérience en programmation requise

Objectifs pédagogiques

  1. Installer et configurer R et RStudio
  2. Naviguer dans RStudio : console, éditeur, environnement, aide
  3. Identifier et créer les objets R : vecteurs, matrices, listes, data.frames, facteurs
  4. Maîtriser les types de données : numeric, character, logical, integer et leurs conversions
  5. Utiliser les fonctions de base : math, stats, chaînes de caractères, logique
  6. Écrire ses propres fonctions pour automatiser des traitements
  7. Importer et exporter des données (CSV, Excel, RDS)
  8. Manipuler un data.frame : sélection, filtrage, tri, jointures, agrégation
  9. Créer des graphiques de base : scatter, histogramme, boxplot, barplot
  10. Mener une analyse exploratoire complète de l'import à la visualisation

Programme

Jour 1 — Découverte de R

Module Contenu
1. Introduction à R et RStudio Installation, interface RStudio (4 panneaux), console, scripts, obtenir de l'aide
2. Les objets R Types atomiques, vecteurs, indexation, matrices, listes, data.frames, facteurs, conversions
3. Les fonctions de base Fonctions math, stats (mean, sd, summary), chaînes (paste, gsub), logique, fonctions custom
4. Atelier pratique Exercices guidés consolidant les modules 1 à 3

Jour 2 — Manipulation de données

Module Contenu
5. Import et export read.csv, read.table, readxl, readRDS, write.csv, exploration (str, head, summary)
6. Manipulation de data.frames Sélection, filtrage, tri, colonnes calculées, merge/jointures, agrégation
7. Visualisation de base plot(), hist(), boxplot(), barplot(), paramètres graphiques, multi-graphiques, export
8. Projet intégrateur Analyse complète sur airquality : import, nettoyage, stats, visualisation
