R Niveau I

R Niveau I — Plan de cours détaillé

Formation : Initiation au langage R Durée : 2 jours (14 heures) Horaires : 9h00 – 17h00

JOUR 1 — Découverte de R

Module 1 : Introduction à R et RStudio (9h00 – 10h30)

Objectifs : Installer R et RStudio, comprendre l'interface, exécuter ses premières commandes.

Contenu détaillé :

  1. Pourquoi R ? (15 min)

    • R dans l'écosystème data science (comparaison Python, SAS, Excel)
    • Open source, communauté active, reproductibilité
    • Cas d'usage : statistique, bioinformatique, finance, sciences sociales

  2. Installation de R et RStudio (15 min)

    • Téléchargement depuis CRAN et posit.co
    • Démonstration de l'installation pas à pas
    • Vérification de l'installation

  3. L'interface RStudio (20 min)

    • Les 4 panneaux : Console, Source, Environment/History, Files/Plots/Help
    • Personnalisation : thème, disposition, raccourcis clavier
    • Créer un nouveau script R

  4. La console R (20 min)

    • R comme calculatrice : opérations arithmétiques
    • Affectation avec <- et =
    • Afficher et inspecter des valeurs
    • L'historique des commandes

  5. Les scripts R (10 min)

    • Créer, sauvegarder, exécuter un script (.R)
    • Commentaires avec #
    • Exécuter ligne par ligne vs bloc

  6. Obtenir de l'aide (10 min)

    • ?fonction, help(fonction), ??mot_clé
    • Onglet Help de RStudio
    • Exemples avec example(mean)

Activité : Créer un script qui effectue 10 calculs variés et les affecte à des variables nommées.

Module 2 : Les objets R (10h45 – 12h15)

Objectifs : Identifier et créer les objets fondamentaux de R, comprendre les types de données.

Contenu détaillé :

  1. Les types atomiques (15 min)

    • numeric (double), integer (suffixe L), character, logical
    • complex (mention rapide)
    • Vérifier un type : class(), typeof(), is.numeric(), is.character()

  2. Les vecteurs (20 min)

    • Création : c(), seq(), rep(), opérateur :
    • Vecteurs nommés : names()
    • Propriété fondamentale : un seul type par vecteur (coercition automatique)
    • Opérations vectorisées : addition, multiplication, recyclage

  3. L'indexation de vecteurs (15 min)

    • Par position : v[1], v[c(1,3)]
    • Par exclusion : v[-2]
    • Par condition logique : v[v > 5]
    • Par nom : v["nom"]

  4. Les matrices (10 min)

    • matrix() : nrow, ncol, byrow
    • Dimensions : dim(), nrow(), ncol()
    • Indexation : m[i, j], m[i, ], m[, j]

  5. Les listes (15 min)

    • list() : conteneur hétérogène
    • Accès : $nom, [[ ]] (élément), [ ] (sous-liste)
    • Listes nommées
    • Quand utiliser une liste vs un vecteur

  6. Les data.frames (10 min)

    • data.frame() : le tableau de données
    • Accès colonnes : df$col, df[, "col"], df[, c("a","b")]
    • Accès lignes : df[1:5, ]
    • str(), head(), summary()

  7. Les facteurs (5 min)

    • factor() : variables catégorielles
    • levels(), facteurs ordonnés
    • Utilité pour les modèles statistiques et les graphiques

  8. Conversions de types (5 min)

    • as.numeric(), as.character(), as.logical(), as.integer()
    • as.data.frame(), as.matrix()
    • Pièges courants de la coercition

Activité : Créer un data.frame de 5 étudiants (nom, âge, note, réussite) et le manipuler.

Module 3 : Fonctions et contrôle de flux (13h30 – 15h00)

Objectifs : Utiliser les fonctions intégrées de R et écrire ses premières fonctions personnalisées.

Contenu détaillé :

  1. Fonctions mathématiques (15 min)

    • sqrt(), abs(), round(), ceiling(), floor()
    • log(), log10(), exp()
    • max(), min(), range(), sum(), prod(), cumsum()

  2. Fonctions statistiques (20 min)

    • mean(), median(), sd(), var()
    • quantile(), IQR()
    • summary() : résumé en une ligne
    • cor() : corrélation entre deux vecteurs
    • table() : tableau de fréquences

  3. Manipulation de chaînes (15 min)

    • paste() et paste0() : concaténation
    • nchar() : longueur
    • substr(), substring() : extraction
    • toupper(), tolower() : casse
    • gsub(), sub() : remplacement
    • grep(), grepl() : recherche de motifs

  4. Fonctions logiques (10 min)

    • any(), all() : au moins un / tous vrais
    • which() : indices des vrais
    • ifelse() : condition vectorisée
    • Opérateurs &, |, !, &&, ||

  5. Créer ses propres fonctions (25 min)

    • Syntaxe : function(arg1, arg2 = default) { ... }
    • return() explicite vs implicite
    • Arguments par défaut
    • Portée des variables (scope)
    • Exemple : fonction qui calcule un z-score

  6. Appliquer une fonction (5 min)

    • sapply(), lapply() : aperçu rapide
    • Appliquer une fonction custom à un vecteur

Activité : Écrire 3 fonctions — calcul d'IMC, standardisation z-score, nettoyage de texte.

Module 4 : Atelier guidé (15h15 – 17h00)

Objectifs : Consolider les acquis des Modules 1 à 3 par la pratique guidée.

Contenu :

  1. Exercice guidé 1 (20 min) — Vecteurs et indexation

    • Créer des vecteurs de températures, les filtrer, calculer des statistiques

  2. Exercice guidé 2 (20 min) — Data.frames

    • Construire un data.frame de ventes, ajouter des colonnes calculées

  3. Exercice guidé 3 (20 min) — Fonctions

    • Écrire une fonction de conversion Celsius/Fahrenheit
    • L'appliquer à un vecteur de mesures

  4. Exercice guidé 4 (25 min) — Mini-analyse

    • Créer un jeu de données complet, calculer des statistiques par groupe

  5. Correction collective et questions (20 min)

JOUR 2 — Manipulation de données

Module 5 : Import / Export (9h00 – 10h30)

Objectifs : Charger des données depuis des sources variées et les explorer.

Contenu détaillé :

  1. read.csv() et read.table() (20 min)

    • Paramètres essentiels : sep, header, dec, stringsAsFactors
    • na.strings : gérer les valeurs manquantes à l'import
    • encoding : UTF-8, Latin1
    • read.csv2() pour les fichiers français (; et ,)

  2. read.xlsx avec le package readxl (10 min)

    • install.packages("readxl") et library(readxl)
    • read_excel() : paramètres sheet, range, col_names

  3. readRDS et saveRDS (10 min)

    • Format natif R : compact, rapide, préserve les types
    • saveRDS(objet, "fichier.rds") / readRDS("fichier.rds")

  4. write.csv et write.table (10 min)

    • row.names = FALSE
    • write.csv2() pour le format français

  5. Exploration initiale (25 min)

    • str() : structure complète
    • head(), tail() : premières / dernières lignes
    • summary() : résumé statistique
    • dim(), nrow(), ncol() : dimensions
    • names(), colnames() : noms de colonnes
    • class() : type de chaque colonne

  6. Gestion des valeurs manquantes (15 min)

    • is.na(), sum(is.na(df))
    • complete.cases()
    • na.rm = TRUE dans les fonctions
    • na.omit() : supprimer les lignes incomplètes

Activité : Importer un CSV avec des problèmes (encodage, séparateur, NA), le nettoyer et l'explorer.

Module 6 : Manipulation de data frames (10h45 – 12h15)

Objectifs : Maîtriser les opérations courantes de manipulation tabulaire en base R.

Contenu détaillé :

  1. Sélection de colonnes (10 min)

    • df$col, df[, "col"], df[, c("a", "b")]
    • Sélection par indice numérique

  2. Filtrage de lignes (15 min)

    • Conditions logiques : df[df$age > 30, ]
    • subset(df, age > 30, select = c(nom, age))
    • Combinaison de conditions : &, |

  3. Tri (10 min)

    • order() : retourne les indices triés
    • df[order(df$note), ] : tri croissant
    • df[order(-df$note), ] : tri décroissant
    • Tri multi-critères

  4. Ajout de colonnes calculées (10 min)

    • df$nouvelle <- df$a + df$b
    • transform(df, ratio = a / b)
    • within(df, { ... })

  5. Merge et jointures (20 min)

    • merge(x, y, by = "clé") : jointure naturelle
    • all.x = TRUE : left join
    • all.y = TRUE : right join
    • all = TRUE : full join
    • Jointure sur plusieurs colonnes : by = c("a", "b")

  6. Reshape (10 min)

    • stack() et unstack() : format long ↔ large
    • reshape() : transformation plus contrôlée

  7. Agrégation (15 min)

    • aggregate(valeur ~ groupe, data = df, FUN = mean)
    • tapply(df$valeur, df$groupe, summary)
    • Agrégation multi-fonctions

Activité : Pipeline complet — filtrer, trier, joindre deux tables, agréger par groupe.

Module 7 : Visualisation (13h30 – 15h00)

Objectifs : Créer des graphiques informatifs avec le système graphique de base de R.

Contenu détaillé :

  1. plot() — Nuage de points (15 min)

    • plot(x, y) : scatterplot de base
    • Paramètres : main, xlab, ylab, col, pch, cex
    • Types : type = "p", "l", "b", "h"

  2. hist() — Histogramme (10 min)

    • breaks : nombre de classes
    • col, border
    • Ajouter une courbe de densité

  3. boxplot() — Boîte à moustaches (10 min)

    • boxplot(valeur ~ groupe) : comparaison par groupe
    • Interprétation : médiane, quartiles, outliers
    • horizontal = TRUE

  4. barplot() — Diagramme en barres (10 min)

    • barplot(table(df$var))
    • Barres empilées et côte à côte
    • legend.text, args.legend

  5. Paramètres graphiques avancés (15 min)

    • par() : marges, couleurs, polices
    • pch (1-25), lty (1-6), col (noms, hex, rgb)
    • cex (taille), lwd (épaisseur)

  6. Combiner des graphiques (10 min)

    • par(mfrow = c(2, 2)) : grille 2×2
    • layout() : disposition personnalisée

  7. Ajouter des éléments (10 min)

    • legend(), abline(), text(), points(), lines()
    • title() pour ajouter des titres

  8. Exporter des graphiques (10 min)

    • pdf("fichier.pdf") ... dev.off()
    • png("fichier.png", width, height, res)
    • Depuis RStudio : Export button

Activité : Créer un dashboard de 4 graphiques sur le jeu de données iris.
