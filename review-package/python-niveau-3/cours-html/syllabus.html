Python Niveau III — Syllabus

NumPy • pandas • Contrôle qualité des données

Durée recommandée : 3 jours (21h) — niveau data intermédiaire avancé.

Objectif général

Passer d'analyses ponctuelles à des pipelines de données reproductibles, contrôlés et exploitables en environnement métier.

Progression pédagogique (du simple au complexe)

Étape 1 — Fondations NumPyComprendre arrays, types, axes et vectorisation.
Étape 2 — Fondations pandasImporter, typer et nettoyer les données tabulaires.
Étape 3 — Transformations métierRegrouper, joindre, pivoter, calculer des KPI fiables.
Étape 4 — Temporalité et robustesseResampling, fenêtres, contrôles de qualité.
Étape 5 — RestitutionExporter des livrables clairs et documenter les hypothèses.

Définitions essentielles

  • Vectorisation : calcul appliqué à un tableau complet sans boucle Python explicite.
  • DataFrame : table de données pandas indexée en lignes/colonnes.
  • Jointure : combinaison de tables via des clés communes.
  • Cardinalité : relation entre clés (one-to-one, one-to-many, etc.).
  • KPI : indicateur métier défini, calculé et suivi dans le temps.

Notions théoriques supplémentaires

  • Copy vs view : impact mémoire et risques d'écriture involontaire sur les données sources.
  • Indexation avancée : différences entre loc, iloc, index hiérarchiques et slicing temporel.
  • Dtypes et coercition : fiabilité numérique, colonnes catégorielles, précision et NA.
  • Fenêtres glissantes : moyenne mobile, lissage, signaux de tendance.
  • Idempotence data : exécuter plusieurs fois un pipeline sans dérive des résultats.
  • Data contracts : schéma attendu, règles métier et contrôles automatiques.

Extraits de code de référence

NumPy : normalisation

import numpy as np
x = np.array([10, 12, 15, 21], dtype=float)
z = (x - x.mean()) / x.std()

pandas : nettoyage

import pandas as pd

df = pd.read_csv("ventes.csv", parse_dates=["date"])
df["prix_net"] = pd.to_numeric(df["prix_net"], errors="coerce")
df = df.dropna(subset=["prix_net", "client_id"])
df = df[df["prix_net"] >= 0]

Contrôle de schéma minimal

expected = {"date": "datetime64[ns]", "region": "object", "prix_net": "float64"}
for col, dtype in expected.items():
    if col not in df.columns:
        raise ValueError(f"colonne manquante: {col}")
    if str(df[col].dtype) != dtype:
        raise ValueError(f"dtype invalide pour {col}: {df[col].dtype}")
