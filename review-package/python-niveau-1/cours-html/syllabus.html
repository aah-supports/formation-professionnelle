Python Niveau I

Python Niveau I — Syllabus

Initiation au langage Python

Durée : 2 jours (14 heures) Format : Présentiel ou distanciel synchrone

Description

Formation de deux jours pour découvrir Python et son écosystème. Les stagiaires acquièrent les bases pour écrire des scripts, manipuler les structures de données fondamentales, lire et écrire des fichiers, et mener une analyse exploratoire de données avec pandas et matplotlib.

Public cible

  • Analystes de données débutants en Python
  • Professionnels migrant depuis Excel, SPSS ou SAS
  • Étudiants en data science, biologie, économie, sciences sociales
  • Professionnels souhaitant automatiser des traitements de données

Prérequis

  • Notions de statistique descriptive (moyenne, médiane, écart-type)
  • Aisance informatique de base
  • Aucune expérience en programmation requise

Objectifs pédagogiques

  1. Installer et configurer Python et un environnement de développement (VS Code, Jupyter)
  2. Utiliser l'interpréteur interactif et écrire des scripts .py
  3. Identifier et manipuler les types de données : int, float, str, bool, None
  4. Maîtriser les structures : listes, tuples, dictionnaires, ensembles
  5. Écrire des fonctions et utiliser les structures de contrôle (if, for, while)
  6. Lire et écrire des fichiers (texte, CSV, JSON)
  7. Importer et utiliser des modules standards et externes (pip)
  8. Calculer avec numpy : tableaux, opérations vectorisées, statistiques
  9. Manipuler des données tabulaires avec pandas : sélection, filtrage, agrégation
  10. Créer des visualisations avec matplotlib : scatter, histogramme, boxplot, barplot
  11. Mener une analyse exploratoire complète de l'import à la visualisation

Programme

Jour 1 — Découverte de Python

Module Contenu
1. Introduction à Python Installation, environnements (VS Code, Jupyter), interpréteur, scripts, pip, aide
2. Types et structures de données int, float, str, bool, listes, tuples, dictionnaires, ensembles, indexation, slicing
3. Fonctions et contrôle if/elif/else, for, while, range, fonctions def, arguments, compréhensions de liste
4. Atelier pratique Exercices guidés consolidant les modules 1 à 3

Jour 2 — Manipulation de données

Module Contenu
5. Fichiers et modules open/read/write, with, csv, json, os.path, import, modules standards
6. NumPy et pandas Tableaux numpy, opérations vectorisées, DataFrame, read_csv, loc/iloc, filtrage, tri, merge, groupby
7. Visualisation avec matplotlib scatter, hist, boxplot, bar, subplots, personnalisation, export
8. Projet intégrateur Analyse complète : import, nettoyage, statistiques, visualisation
