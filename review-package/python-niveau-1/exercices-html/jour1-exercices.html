Exercice 2.5 : Tuples [Difficulté : Intermédiaire]

Énoncé :

  1. Creez un tuple coordonnees = (48.8566, 2.3522) (latitude, longitude de Paris)
  2. Accedez a la latitude et la longitude par indexation
  3. Utilisez le unpacking : lat, lon = coordonnees
  4. Creez un tuple de tuples representant 3 villes avec leurs coordonnees
  5. Montrez qu'un tuple est immutable (essayez de modifier un element)
  6. Utilisez un tuple comme cle de dictionnaire

Indice : Les tuples sont immutables : on ne peut pas modifier leurs elements. C'est pour cela qu'on peut les utiliser comme cles de dictionnaire (contrairement aux listes).

Solution 
# 1. Creation
coordonnees = (48.8566, 2.3522)
print(f"Coordonnees : {coordonnees}")

# 2. Acces par index
print(f"Latitude  : {coordonnees[0]}")
print(f"Longitude : {coordonnees[1]}")

# 3. Unpacking
lat, lon = coordonnees
print(f"lat = {lat}, lon = {lon}")

# 4. Tuple de tuples
villes = (
    ("Paris", 48.8566, 2.3522),
    ("Lyon", 45.7640, 4.8357),
    ("Marseille", 43.2965, 5.3698),
)
for nom, lat, lon in villes:
    print(f"{nom} : {lat}, {lon}")

# 5. Immutabilite
try:
    coordonnees[0] = 0
except TypeError as e:
    print(f"\nErreur : {e}")

# 6. Tuple comme cle de dictionnaire
distances = {
    ("Paris", "Lyon"): 465,
    ("Paris", "Marseille"): 775,
    ("Lyon", "Marseille"): 315,
}
print(f"\nParis-Lyon : {distances[('Paris', 'Lyon')]} km")

Exercice 2.8 : Conversions et mutabilité [Difficulté : Intermédiaire]

Énoncé :

  1. Creez une liste, un tuple et un set contenant les memes elements [1, 2, 3, 4, 5]
  2. Convertissez la liste en tuple, le tuple en set, le set en liste
  3. Montrez que la liste est mutable (modifiez un element)
  4. Montrez que le tuple est immutable (essayez de modifier un element)
  5. Montrez qu'un set ne garde pas l'ordre et n'accepte pas les doublons
  6. Creez une copie superficielle d'une liste et montrez la difference avec =

Indice : list(), tuple(), set() permettent les conversions. Pour copier une liste : liste.copy() ou liste[:]. L'assignation b = a ne copie pas, elle cree une reference.

Solution 
# 1. Trois structures
ma_liste = [1, 2, 3, 4, 5]
mon_tuple = (1, 2, 3, 4, 5)
mon_set = {1, 2, 3, 4, 5}

print(f"Liste : {ma_liste} (type: {type(ma_liste).__name__})")
print(f"Tuple : {mon_tuple} (type: {type(mon_tuple).__name__})")
print(f"Set   : {mon_set} (type: {type(mon_set).__name__})")

# 2. Conversions
print(f"\nListe -> Tuple : {tuple(ma_liste)}")
print(f"Tuple -> Set   : {set(mon_tuple)}")
print(f"Set   -> Liste : {sorted(list(mon_set))}")

# 3. Liste mutable
ma_liste[0] = 99
print(f"\nListe modifiee : {ma_liste}")

# 4. Tuple immutable
try:
    mon_tuple[0] = 99
except TypeError as e:
    print(f"Erreur tuple : {e}")

# 5. Set : pas d'ordre, pas de doublons
s = {3, 1, 4, 1, 5, 9, 2, 6, 5, 3}
print(f"\nSet avec doublons : {s}")

# 6. Copie vs reference
a = [1, 2, 3]
b = a           # reference (meme objet)
c = a.copy()    # copie superficielle

a[0] = 999
print(f"\na (modifie) : {a}")
print(f"b (reference) : {b}")    # modifie aussi !
print(f"c (copie)     : {c}")    # non modifie

  4. Testez chaque fonction avec plusieurs valeurs

Énoncé :

  2. Ecrivez une fonction puissance(base, exposant=2) qui calcule base^exposant

  4. Ecrivez une fonction fiche(**kwargs) qui affiche toutes les paires cle-valeur passees en argument

Indice : *args recoit les arguments positionnels en tuple. **kwargs recoit les arguments nommes en dictionnaire.

# 1. Argument par defaut
def saluer(nom, salutation="Bonjour"):
    return f"{salutation}, {nom} !"

    return base ** exposant

# 3. Moyenne avec *args
def moyenne(*args):
    if len(args) == 0:
        return 0
    return sum(args) / len(args)

print(f"moyenne(5) = {moyenne(5)}")
print(f"moyenne(1, 2, 3, 4, 5) = {moyenne(1, 2, 3, 4, 5)}")

# 4. Fiche avec **kwargs
def fiche(**kwargs):
    print("--- Fiche ---")

Exercice 3.7 : Fonctions avancées [Difficulté : Avancé]

Énoncé :

  1. Creez une fonction lambda qui calcule le carre d'un nombre et testez-la
  2. Utilisez map() pour convertir une liste de temperatures Celsius en Fahrenheit
  3. Utilisez filter() pour garder uniquement les nombres positifs de [-3, 7, -1, 4, -8, 2, 0, 5]
  4. Utilisez sorted() avec le parametre key pour trier une liste de tuples ("nom", age) par age
  5. Combinez map() et filter() : a partir d'une liste de nombres, gardez les pairs et calculez leurs carres
  6. Utilisez sorted() avec une lambda pour trier des mots par leur derniere lettre

Indice : lambda x: expression cree une fonction anonyme. map(fonction, iterable) applique une fonction. filter(fonction, iterable) filtre. sorted(iterable, key=fonction) trie.

Solution 
# 1. Lambda
carre = lambda x: x ** 2
print(f"carre(5) = {carre(5)}")
print(f"carre(12) = {carre(12)}")

# 2. map() : Celsius -> Fahrenheit
celsius = [0, 10, 20, 25, 30, 37, 100]
fahrenheit = list(map(lambda c: c * 9/5 + 32, celsius))
print(f"\nCelsius    : {celsius}")
print(f"Fahrenheit : {fahrenheit}")

# 3. filter() : nombres positifs
nombres = [-3, 7, -1, 4, -8, 2, 0, 5]
positifs = list(filter(lambda x: x > 0, nombres))
print(f"\nNombres  : {nombres}")
print(f"Positifs : {positifs}")

# 4. sorted() avec key : trier par age
personnes = [("Alice", 30), ("Bob", 22), ("Clara", 27), ("David", 19)]
par_age = sorted(personnes, key=lambda p: p[1])
print(f"\nTrie par age : {par_age}")

# 5. Combiner filter + map
nombres = list(range(1, 21))
pairs_carres = list(map(lambda x: x ** 2, filter(lambda x: x % 2 == 0, nombres)))
print(f"\nCarres des pairs de 1-20 : {pairs_carres}")

# 6. Trier par derniere lettre
mots = ["banane", "pomme", "kiwi", "fraise", "ananas"]
par_derniere = sorted(mots, key=lambda m: m[-1])
print(f"\nTrie par derniere lettre : {par_derniere}")
