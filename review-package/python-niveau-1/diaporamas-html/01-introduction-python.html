Installation de Python

Télécharger et configurer en cinq minutes

  • Télécharger depuis python.org (version 3.12+)
  • Windows : cocher « Add Python to PATH »
  • Mac : python.org ou brew install python
  • Linux : souvent préinstallé
  • Vérifier : python --version

Les scripts Python

Écrire pour reproduire

# mon_premier_script.py
# Analyse des températures

temps = [18, 22, 25, 19, 30]
moyenne = sum(temps) / len(temps)
print(f"Moyenne : {moyenne}°C")
# Moyenne : 22.8°C

Bonnes pratiques

Commencer du bon pied

  • Environnements virtuels : python -m venv mon_env

  • Commenter le pourquoi, pas le quoi

  • Sauvegarder souvent, versionner avec Git
