Exercice 2 : Listes

Températures d'une semaine

temps = [18, 22, 25, 19, 30, 17, 21]

# Statistiques de base
print(f"Max : {max(temps)}°C")       # 30
print(f"Min : {min(temps)}°C")       # 17
moyenne = sum(temps) / len(temps)
print(f"Moyenne : {moyenne:.1f}°C")  # 21.7

# Jours chauds (compréhension)
chauds = [t for t in temps if t > 20]
print(f"{len(chauds)} jours > 20°C")  # 4

# Top 3
temps_tri = sorted(temps, reverse=True)
top3 = temps_tri[:3]  # [30, 25, 22]

Exercice 4 : Fonctions

Trois fonctions utilitaires

def celsius_to_f(c):
    return c * 9/5 + 32

def stats(nombres):
    return {
        "min": min(nombres),
        "max": max(nombres),
        "moy": sum(nombres) / len(nombres)
    }

def est_palindrome(mot):
    mot = mot.lower().replace(" ", "")
    return mot == mot[::-1]

print(celsius_to_f(100))       # 212.0
print(stats([14, 8, 17, 12]))  # {min:8, max:17, moy:12.75}
print(est_palindrome("Kayak")) # True

Exercice 5 : Mini-analyse

Combiner toutes les compétences

etudiants = [
    {"nom": "Léa",  "classe": "A", "note": 14},
    {"nom": "Tom",  "classe": "B", "note": 12},
    {"nom": "Aya",  "classe": "A", "note": 17},
    {"nom": "Hugo", "classe": "B", "note": 15},
    {"nom": "Noa",  "classe": "A", "note": 11},
    {"nom": "Mia",  "classe": "B", "note": 18}
]

def moyenne_par_classe(data):
    classes = {}
    for e in data:
        classes.setdefault(e["classe"], []).append(e["note"])
    return {c: sum(n)/len(n) for c, n in classes.items()}

print(moyenne_par_classe(etudiants))
# {"A": 14.0, "B": 15.0}

Les pièges à éviter

  • Indentation : toujours 4 espaces, jamais de mélange
  • str.upper() renvoie une copie, ne modifie pas l'original
  • Comparer à None avec 'is', pas '=='
  • dict.get(clé, défaut) évite les KeyError
  • sorted() renvoie une copie, .sort() modifie sur place
