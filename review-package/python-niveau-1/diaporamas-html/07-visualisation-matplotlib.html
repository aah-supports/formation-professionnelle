Module 7 — Visualisation avec matplotlib

Construire des graphiques lisibles à partir de listes Python.

Objectifs

  • Choisir le bon type de graphique.
  • Créer des visuels clairs et interprétables.
  • Éviter les biais de présentation.
  • Exporter un graphique prêt à partager.

Ligne temporelle

import matplotlib.pyplot as plt

months = ["Jan", "Fév", "Mar", "Avr", "Mai"]
ca = [1200, 1350, 1280, 1490, 1610]

plt.plot(months, ca, marker="o")
plt.title("CA mensuel")
plt.xlabel("Mois")
plt.ylabel("CA (€)")
plt.show()

Barres comparatives

import matplotlib.pyplot as plt

canaux = ["Web", "Boutique", "Partenaire"]
ventes = [320, 180, 95]

plt.bar(canaux, ventes)
plt.title("Ventes par canal")
plt.ylabel("Nombre")
plt.show()

Histogramme

import matplotlib.pyplot as plt

notes = [10, 12, 14, 14, 15, 15, 16, 17, 18, 12, 11, 13]

plt.hist(notes, bins=6, edgecolor="black")
plt.title("Distribution des notes")
plt.xlabel("Note")
plt.ylabel("Fréquence")
plt.show()

Nuage de points

import matplotlib.pyplot as plt

x = [1, 2, 3, 4, 5, 6]
y = [2, 4, 5, 4, 6, 7]

plt.scatter(x, y)
plt.title("Relation X/Y")
plt.xlabel("X")
plt.ylabel("Y")
plt.show()

Personnalisation utile

  • Titre explicite orienté métier.
  • Axes nommés avec unité.
  • Légende dès qu'il y a plusieurs séries.
  • Palette sobre et cohérente.

Sous-graphiques

import matplotlib.pyplot as plt

fig, axes = plt.subplots(1, 2, figsize=(10, 4))
axes[0].plot([1, 2, 3], [2, 3, 5])
axes[0].set_title("Tendance")
axes[1].bar(["A", "B", "C"], [4, 7, 5])
axes[1].set_title("Comparaison")
plt.tight_layout()
plt.show()

Exporter un visuel

plt.figure(figsize=(8, 4))
plt.plot([1, 2, 3], [3, 2, 4])
plt.title("Évolution")
plt.tight_layout()
plt.savefig("evolution.png", dpi=150)

Erreurs fréquentes

  • Axes tronqués sans le signaler.
  • Trop de catégories sur un seul graphe.
  • Couleurs non lisibles (faible contraste).
  • Graphique sans contexte ni source.

Récapitulatif

  • Un graphique est utile s'il répond à une question précise.
  • La lisibilité est prioritaire sur l'esthétique.
  • Matplotlib suffit pour des dashboards pédagogiques de base.
