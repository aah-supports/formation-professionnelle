Projet fil rouge — R Niveau II

Exploiter une base SQL et des exports Excel pour un reporting descriptif

Durée : 1h45 Sources : base SQL (business.db) + classeurs Excel (catalogue.xlsx, objectifs.xlsx) Livrable : script R reproductible + note de synthèse descriptive

Contexte métier

Vous accompagnez l'équipe commerciale d'une entreprise multi-régions. Les ventes détaillées sont stockées en base SQL, mais les objectifs et certaines données produits sont diffusés sous Excel. Le management demande un reporting fiable avec focus descriptif :

  • où le chiffre d'affaires progresse / recule,
  • quels segments client tirent la performance,
  • quels écarts existent entre réalisé et objectif.

Étape 1 : Extraction multi-sources (20 min)

  1. Connexion à la base avec DBI::dbConnect()
  2. Lecture des tables SQL ventes et clients en mode distant
  3. Import des fichiers Excel catalogue.xlsx et objectifs.xlsx
  4. Nettoyage minimal (clean_names(), typage, valeurs manquantes évidentes)

Attendu : 4 objets propres prêts à être joints.

Étape 2 : Contrôles qualité (20 min)

  1. Vérifier unicité de la clé id_commande
  2. Calculer le taux de NA des colonnes critiques (date, client_id, prix_net)
  3. Détecter les produits vendus mais absents du catalogue (anti_join)
  4. Produire un mini tableau de contrôle (OK / Alerte)

Attendu : un bloc qualité explicite avant toute analyse.

Étape 3 : Requêtes avancées et table analytique (25 min)

  1. Construire une table unifiée (ventes + clients + produits)
  2. Calculer montant_commande = quantite * prix_net
  3. Agréger par mois, région, segment
  4. Joindre les objectifs Excel par mois/région
  5. Calculer ecart_obj et taux_atteinte

Attendu : table finale de pilotage avec métriques métier.

Étape 4 : Statistiques descriptives pratiques (25 min)

Produire une synthèse descriptive actionnable :

  1. 3 KPI globaux : CA total, panier moyen, nombre de clients actifs
  2. Descriptif par segment : médiane panier, dispersion (IQR), part de CA
  3. Évolution temporelle : variation MoM (%) et cumul annuel
  4. Top 5 clients par région avec ranking

Attendu : interprétation courte de chaque résultat (pas seulement des chiffres).

Étape 5 : Restitution (15 min)

  1. Créer un tableau final avec gt (ou tableau dplyr propre)
  2. Créer 2 graphiques descriptifs utiles (ligne temporelle + barres segment)
  3. Exporter les résultats (CSV/Excel + PNG)
  4. Rédiger une synthèse de 8 à 10 lignes : constats + recommandations

library(DBI)
library(RSQLite)
library(dplyr)
library(dbplyr)
library(readxl)
library(janitor)
library(gt)

# 1) Extraction
con <- dbConnect(SQLite(), "business.db")
ventes_tbl  <- tbl(con, "ventes")
clients_tbl <- tbl(con, "clients")

catalogue <- read_excel("catalogue.xlsx", sheet = "Produits") |> clean_names()
objectifs <- read_excel("objectifs.xlsx", sheet = "Objectifs", skip = 2) |>
  clean_names()

# 2) Contrôles qualité (exemples)
qc_na <- ventes_tbl |>
  summarise(
    na_date = sum(is.na(date)),
    na_client = sum(is.na(client_id)),
    na_prix = sum(is.na(prix_net))
  ) |> collect()

produits_absents <- ventes_tbl |>
  distinct(produit_id) |>
  collect() |>
  anti_join(catalogue |> distinct(produit_id), by = "produit_id")

# 3) Table analytique
pilotage <- ventes_tbl |>
  left_join(clients_tbl, by = "client_id") |>
  collect() |>
  left_join(catalogue, by = "produit_id") |>
  mutate(
    mois = as.Date(format(date, "%Y-%m-01")),
    montant_commande = quantite * prix_net
  ) |>
  group_by(mois, region, segment) |>
  summarise(
    ca = sum(montant_commande, na.rm = TRUE),
    clients_actifs = n_distinct(client_id),
    panier_moyen = mean(montant_commande, na.rm = TRUE),
    .groups = "drop"
  ) |>
  left_join(objectifs, by = c("mois", "region")) |>
  mutate(
    ecart_obj = ca - objectif_ca,
    taux_atteinte = ca / objectif_ca
  )

# 4) Descriptif segment
desc_segment <- pilotage |>
  group_by(segment) |>
  summarise(
    ca_total = sum(ca, na.rm = TRUE),
    mediane_panier = median(panier_moyen, na.rm = TRUE),
    iqr_panier = IQR(panier_moyen, na.rm = TRUE),
    .groups = "drop"
  ) |>
  mutate(part_ca = ca_total / sum(ca_total))

# 5) Restitution
desc_segment |>
  gt() |>
  fmt_currency(columns = c(ca_total, mediane_panier), currency = "EUR") |>
  fmt_percent(columns = part_ca)

dbDisconnect(con)
