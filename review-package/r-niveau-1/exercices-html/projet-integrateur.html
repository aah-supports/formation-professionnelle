Analyse de la qualité de l'air à New York

Durée : 1h45 Jeu de données : airquality (intégré dans R) Livrable : Un script R commenté (.R)

Contexte

Vous êtes analyste dans une agence environnementale. On vous confie les mesures de qualité de l'air à New York durant l'été 1973. Votre mission : explorer, nettoyer et analyser ces données pour produire un rapport visuel et chiffré.

Le jeu de données airquality contient 153 observations et 6 variables :

  • Ozone : concentration d'ozone (ppb)
  • Solar.R : rayonnement solaire (lang)
  • Wind : vitesse du vent (mph)
  • Temp : température (°F)
  • Month : mois (5 = mai, 9 = septembre)
  • Day : jour du mois

Étape 1 : Chargement et exploration (20 min)

  1. Chargez le jeu de données avec data(airquality)
  2. Affichez la structure avec str()
  3. Affichez les 10 premières lignes avec head()
  4. Obtenez les dimensions avec dim()
  5. Produisez un résumé statistique avec summary()
  6. Comptez le nombre de valeurs manquantes par variable avec sapply() et is.na()

Questions :

  • Combien de lignes et colonnes ?
  • Quelles variables ont des NA ? Combien ?
  • Quelle est la plage de températures observées ?

Étape 2 : Nettoyage (15 min)

  1. Créez une copie du data.frame : aq <- airquality
  2. Supprimez les lignes où Ozone est NA
  3. Vérifiez le nombre de lignes restantes
  4. Ajoutez une colonne Mois_nom avec les noms des mois en français (Mai, Juin, Juillet, Août, Septembre)
  5. Ajoutez une colonne Cat_temp : "Chaud" si Temp > 80, "Tempéré" sinon
  6. Convertissez Mois_nom en facteur ordonné

Indice : Utilisez un vecteur nommé pour la correspondance mois → nom, et ifelse() pour la catégorie.

Étape 3 : Statistiques descriptives (20 min)

  1. Calculez la moyenne, médiane et écart-type de l'ozone
  2. Calculez la moyenne d'ozone par mois avec aggregate()
  3. Calculez la corrélation entre Temp et Ozone
  4. Calculez la corrélation entre Wind et Ozone
  5. Produisez un tableau croisé Cat_temp × Month avec table()
  6. Interprétez : quel mois a le plus d'ozone ? Y a-t-il une relation température-ozone ?

Étape 4 : Visualisations (30 min)

Créez un dashboard de 4 graphiques avec par(mfrow = c(2, 2)) :

  1. Nuage de points : Temp vs Ozone, avec droite de régression (abline(lm(...)))

    • Couleur : "steelblue", points pleins (pch = 19)
    • Titre : "Température vs Ozone"

Exportez le dashboard en PNG (800×600, 150 dpi).

Rédigez 5 conclusions en commentaires R (#) à la fin de votre script :

# ============================================
# PROJET INTÉGRATEUR — R NIVEAU I
# Analyse de la qualité de l'air à New York
# ============================================

# --- ÉTAPE 1 : EXPLORATION ---

data(airquality)

str(airquality)
# 153 obs. of 6 variables

head(airquality, 10)
dim(airquality)   # 153 x 6
summary(airquality)

# Valeurs manquantes par variable
sapply(airquality, function(x) sum(is.na(x)))
# Ozone: 37, Solar.R: 7, Wind: 0, Temp: 0, Month: 0, Day: 0

# Réponses :
# - 153 lignes, 6 colonnes
# - Ozone (37 NA) et Solar.R (7 NA)
# - Température : 56°F à 97°F

# --- ÉTAPE 2 : NETTOYAGE ---

aq <- airquality[!is.na(airquality$Ozone), ]
nrow(aq)  # 116 lignes

# Noms de mois
mois_noms <- c("5" = "Mai", "6" = "Juin", "7" = "Juillet",
               "8" = "Août", "9" = "Septembre")
aq$Mois_nom <- mois_noms[as.character(aq$Month)]

# Catégorie température
aq$Cat_temp <- ifelse(aq$Temp > 80, "Chaud", "Tempéré")

# Facteur ordonné
aq$Mois_nom <- factor(aq$Mois_nom,
  levels = c("Mai", "Juin", "Juillet", "Août", "Septembre"),
  ordered = TRUE)

# --- ÉTAPE 3 : STATISTIQUES DESCRIPTIVES ---

mean(aq$Ozone)     # ~42.1
median(aq$Ozone)   # 31.5
sd(aq$Ozone)       # ~33.0

# Moyenne par mois
aggregate(Ozone ~ Mois_nom, data = aq, FUN = mean)
# Juillet et Août ont les moyennes les plus élevées

# Corrélations
cor(aq$Temp, aq$Ozone)    # ~0.70 (positive forte)
cor(aq$Wind, aq$Ozone)    # ~-0.60 (négative modérée)

# Tableau croisé
table(aq$Cat_temp, aq$Mois_nom)

# --- ÉTAPE 4 : VISUALISATIONS ---

png("dashboard_airquality.png", width = 800, height = 600, res = 150)
par(mfrow = c(2, 2), mar = c(4, 4, 2, 1))

# 1. Scatter
plot(aq$Temp, aq$Ozone, pch = 19, col = "steelblue",
     main = "Température vs Ozone",
     xlab = "Température (°F)", ylab = "Ozone (ppb)")
abline(lm(Ozone ~ Temp, data = aq), col = "red", lwd = 2)

# 2. Histogramme
hist(aq$Ozone, breaks = 15, col = "#a78bfa", border = "white",
     main = "Distribution de l'ozone", xlab = "Ozone (ppb)")

# 3. Boxplot
boxplot(Ozone ~ Month, data = aq,
        col = c("#00f3ff", "#a78bfa", "#ff6b00", "#10b981", "#ff003c"),
        main = "Ozone par mois", xlab = "Mois", ylab = "Ozone (ppb)")

# 4. Barplot
barplot(table(aq$Cat_temp),
        col = c("#ff6b00", "#10b981"),
        main = "Observations par catégorie",
        ylab = "Nombre")

par(mfrow = c(1, 1))
dev.off()

# 1. Forte corrélation positive (r ≈ 0.70) entre température et ozone :
#    les journées chaudes concentrent plus d'ozone troposphérique.
#
# 2. Pic saisonnier en juillet-août, cohérent avec l'ensoleillement
#    maximal qui favorise la formation photochimique d'ozone.
#
# 3. Le vent a un effet dispersant (r ≈ -0.60) : les jours venteux
#    présentent des niveaux d'ozone plus bas.
#
# 4. 24% des mesures d'ozone sont manquantes, ce qui pourrait
#    introduire un biais si les NA ne sont pas aléatoires.
#

#    Ozone ~ Temp + Wind + Solar.R pour quantifier l'effet
