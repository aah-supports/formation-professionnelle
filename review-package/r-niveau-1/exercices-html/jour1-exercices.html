Exercice 3.7 : Appliquer des fonctions avec apply/sapply/lapply [Difficulté : Avancé]

  1. Créez une matrice 4x5 de nombres aléatoires entre 1 et 100 (utilisez set.seed(42) pour la reproductibilité)

  3. Utilisez apply() pour calculer l'écart-type de chaque colonne
  4. Soit nombres <- list(a = 1:10, b = 20:30, c = 50:100). Utilisez sapply() pour calculer la moyenne de chaque élément
  5. Utilisez lapply() pour obtenir le résumé (summary) de chaque élément
  6. Utilisez sapply() avec une fonction anonyme pour calculer le coefficient de variation (sd/mean x 100) de chaque élément

Indice : apply(X, MARGIN, FUN) où MARGIN=1 pour les lignes, MARGIN=2 pour les colonnes. sapply() simplifie le résultat, lapply() retourne une liste.

Solution 
# 1. Matrice aléatoire
set.seed(42)
m <- matrix(sample(1:100, 20, replace = TRUE), nrow = 4, ncol = 5)

# 4. Moyennes avec sapply

sapply(nombres, mean)

# 5. Résumé avec lapply
lapply(nombres, summary)

# 6. Coefficient de variation
sapply(nombres, function(x) sd(x) / mean(x) * 100)
