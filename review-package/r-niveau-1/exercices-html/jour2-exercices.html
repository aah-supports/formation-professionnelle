Exercice 5.1 : Importer un fichier CSV [Difficulté : Facile]

Énoncé :

  1. Créez un petit fichier CSV en R avec write.csv() contenant des données de ventes :
    • produit : c("Pomme", "Banane", "Orange", "Fraise", "Kiwi")
    • prix : c(2.50, 1.80, 3.20, 4.50, 2.90)
    • quantite : c(100, 150, 80, 60, 120)
  2. Réimportez ce fichier avec read.csv()
  3. Vérifiez la structure avec str()
  4. Essayez read.csv2() et expliquez la différence

Indice : write.csv() écrit un CSV avec , comme séparateur. read.csv2() utilise ; comme séparateur (format européen). Pensez à row.names = FALSE pour éviter les numéros de ligne.

Solution 
# 1. Création du CSV
ventes <- data.frame(
  produit = c("Pomme", "Banane", "Orange", "Fraise", "Kiwi"),
  prix = c(2.50, 1.80, 3.20, 4.50, 2.90),
  quantite = c(100, 150, 80, 60, 120)
)
write.csv(ventes, "ventes.csv", row.names = FALSE)

# 2. Réimportation
ventes_import <- read.csv("ventes.csv")
ventes_import

# 3. Structure
str(ventes_import)

# 4. read.csv2 utilise ; comme séparateur et , comme décimale
# Pour un fichier créé avec write.csv, cela ne fonctionnera pas correctement
# car le séparateur est , et non ;
# On crée un fichier au format européen :
write.csv2(ventes, "ventes_eu.csv", row.names = FALSE)
ventes_eu <- read.csv2("ventes_eu.csv")
str(ventes_eu)

Exercice 5.5 : Gérer les valeurs manquantes [Difficulté : Intermédiaire]

Énoncé : Chargez le jeu de données airquality (intégré dans R) :

  1. Combien de valeurs manquantes y a-t-il au total ?
  2. Combien de NA par colonne ?
  3. Quel pourcentage de lignes sont complètes (sans aucun NA) ?
  4. Créez une version du data frame sans aucune ligne contenant des NA
  5. Créez une version où les NA de Ozone sont remplacés par la médiane de Ozone
  6. Comparez les moyennes d'Ozone avant et après imputation

Indice : complete.cases() retourne un vecteur logique. colSums(is.na()) compte les NA par colonne. median(x, na.rm = TRUE) calcule la médiane en ignorant les NA.

Solution 
# 1. Total de NA
sum(is.na(airquality))

# 2. NA par colonne
colSums(is.na(airquality))

# 3. Pourcentage de lignes complètes
pct_complet <- sum(complete.cases(airquality)) / nrow(airquality) * 100
cat("Lignes complètes :", round(pct_complet, 1), "%\n")

# 4. Data frame sans NA
aq_complet <- na.omit(airquality)
cat("Dimensions originales :", dim(airquality), "\n")
cat("Dimensions sans NA :", dim(aq_complet), "\n")

# 5. Imputation par la médiane
aq_impute <- airquality
mediane_ozone <- median(aq_impute$Ozone, na.rm = TRUE)
cat("Médiane Ozone :", mediane_ozone, "\n")
aq_impute$Ozone[is.na(aq_impute$Ozone)] <- mediane_ozone

# 6. Comparaison
cat("Moyenne Ozone (originale, sans NA) :", mean(airquality$Ozone, na.rm = TRUE), "\n")
cat("Moyenne Ozone (après imputation) :", mean(aq_impute$Ozone), "\n")

Exercice 6.5 : Fusion de data frames (merge) [Difficulté : Intermédiaire]

Énoncé :

  1. Créez un data frame employes : id (1:5), nom, departement
  2. Créez un data frame salaires : id (1:6, incluant un id 6 inexistant dans employes), salaire, bonus
  3. Effectuez un inner join (merge par défaut)
  4. Effectuez un left join (garder tous les employés)
  5. Effectuez un full join (garder toutes les lignes)
  6. Fusionnez sur le nom du département avec un 3e data frame

Indice : merge(x, y, by = "colonne") fait un inner join. Ajoutez all.x = TRUE pour un left join, all = TRUE pour un full join.

Solution 
# 1. Employés
employes <- data.frame(
  id = 1:5,
  nom = c("Dupont", "Martin", "Bernard", "Petit", "Robert"),
  departement = c("RH", "IT", "Finance", "IT", "RH")
)

# 2. Salaires
salaires <- data.frame(
  id = c(1, 2, 3, 4, 5, 6),
  salaire = c(3500, 4200, 3800, 4000, 3600, 4500),
  bonus = c(500, 800, 600, 750, 500, 900)
)

# 3. Inner join
inner <- merge(employes, salaires, by = "id")
inner

# 4. Left join
left <- merge(employes, salaires, by = "id", all.x = TRUE)
left

# 5. Full join
full <- merge(employes, salaires, by = "id", all = TRUE)
full

# 6. Fusion sur département
dept_info <- data.frame(
  departement = c("RH", "IT", "Finance", "Marketing"),

Exercice 6.6 : Agrégation de données [Difficulté : Intermédiaire]

Énoncé : Avec mtcars :

  1. Calculez la consommation moyenne par nombre de cylindres avec aggregate()
  2. Calculez la puissance moyenne ET l'écart-type par nombre de cylindres
  3. Calculez le poids moyen par combinaison cylindres/transmission
  4. Comptez le nombre de voitures par catégorie de cylindres
  5. Trouvez la voiture la plus puissante dans chaque groupe de cylindres

Indice : aggregate(formule, data, FUN). La formule est variable ~ groupe. Pour plusieurs variables : cbind(v1, v2) ~ groupe. Pour plusieurs groupes : variable ~ g1 + g2.

Solution 
# 1. Consommation moyenne par cylindres
aggregate(mpg ~ cyl, data = mtcars, FUN = mean)

Exercice 6.7 : Restructurer avec stack/unstack [Difficulté : Intermédiaire]

Énoncé :

  1. Créez un data frame avec les notes de 3 matières pour 4 élèves
  2. Utilisez stack() pour passer en format long (une seule colonne de notes)
  3. Utilisez unstack() pour revenir au format large
  4. Utilisez reshape() pour transformer le data frame de large en long, puis de long en large

Indice : stack() empile les colonnes numériques. reshape() est plus flexible avec les paramètres direction, varying, v.names, timevar, idvar.

Solution

notes <- data.frame(
  maths = c(14, 12, 16, 10),
  physique = c(15, 11, 13, 14),
  chimie = c(12, 14, 17, 11),

)
notes

names(notes_long) <- c("note", "matiere")
notes_long$eleve <- rep(rownames(notes), 3)

notes_large <- unstack(notes_long, note ~ matiere)
notes_large

notes$eleve <- rownames(notes)

                       direction = "long",
                       varying = c("maths", "physique", "chimie"),
                       v.names = "note",
                       timevar = "matiere",
                       times = c("maths", "physique", "chimie"),
                       idvar = "eleve")
notes_long2

# long -> large
notes_large2 <- reshape(notes_long2,
                        direction = "wide",
                        idvar = "eleve",
                        timevar = "matiere")
notes_large2
