read.csv() et read.table()

Les classiques de l'import

# CSV standard (virgule)
df <- read.csv("donnees.csv", fileEncoding = "UTF-8")

# CSV français (point-virgule, décimale virgule)
df <- read.csv2("donnees_fr.csv", fileEncoding = "UTF-8")

# Personnalisé
df <- read.table("donnees.txt",
  sep = "\t",       # tabulation
  header = TRUE,
  dec = ",",
  na.strings = c("", "NA", "N/A"),
  fileEncoding = "UTF-8"
)

Encodage et pièges courants

Éviter les erreurs classiques

  • Encodage : encoding = "UTF-8" ou "latin1"
  • Séparateur : sep = "," ou ";" ou "\t"
  • Répertoire : getwd() / setwd() / projets RStudio
  • stringsAsFactors = FALSE (recommandé)
